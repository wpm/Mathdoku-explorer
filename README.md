# Mathdoku

[![CI](https://github.com/wpm/Mathdoku/actions/workflows/ci.yml/badge.svg)](https://github.com/wpm/Mathdoku/actions/workflows/ci.yml)
[![codecov](https://codecov.io/gh/wpm/Mathdoku/branch/main/graph/badge.svg)](https://codecov.io/gh/wpm/Mathdoku)

A Rust library and desktop designer for creating and solving [Mathdoku](https://en.wikipedia.org/wiki/KenKen) (KenKen) puzzles.

## Crates

- **`mathdoku`** — core library: puzzle representation, constraint propagation, solver, and generator.
- **`mathdoku-designer`** — desktop application for designing and editing Mathdoku puzzles.
